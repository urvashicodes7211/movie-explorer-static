# 🎬 Movie Explorer - Phase 1

A responsive Movie Explorer website developed using **HTML**, **CSS**, and **Bootstrap 5** as part of the Frontend Web Technology (FEWT) Phase-1 project.

## 📌 Project Description

Movie Explorer is a static movie browsing website UI that allows users to explore different movie categories through an attractive and responsive interface.

**Note:** This is the **Phase-1** version of the project. No API integration or backend functionality has been implemented yet.

## 🚀 Features

- Responsive Navigation Bar
- Hero Banner Section
- Trending Movies Section
- Popular Movies Section
- Top Rated Movies Section
- Upcoming Movies Section
- Movie Search Page UI
- Movie Details Page UI
- Watchlist Page UI
- Responsive Design using Bootstrap 5
- Clean and Modern User Interface

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
